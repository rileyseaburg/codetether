## What is CodeTether?

CodeTether is a platform for AI-powered software development, built on the [A2A (Agent-to-Agent) protocol](https://github.com/google/a2a). It provides:

- **CodeTether Agent** - A high-performance Rust CLI that connects to the platform as a worker
- **A2A Server** - Orchestrates tasks between human users and AI agents
- **Multi-Provider AI** - Use OpenAI, Anthropic, Google, DeepSeek, and more
- **Enterprise Security** - All secrets managed via HashiCorp Vault

## Get Started

<Steps>
  <Step title="Install the Agent">
    ```bash
    cargo install codetether-agent
    ```
  </Step>
  <Step title="Configure Vault">
    Store your API keys securely in HashiCorp Vault.
  </Step>
  <Step title="Connect to Platform">
    ```bash
    codetether --server https://api.codetether.run
    ```
  </Step>
</Steps>
