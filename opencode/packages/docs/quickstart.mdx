## Prerequisites

- **Rust** 1.85+ (for building from source) or download the binary
- **HashiCorp Vault** for secrets management
- API key from at least one AI provider (OpenAI, Anthropic, etc.)

## Installation

<Tabs>
  <Tab title="Cargo">
    ```bash
    cargo install codetether-agent
    ```
  </Tab>
  <Tab title="From Source">
    ```bash
    git clone https://github.com/codetether/codetether-agent
    cd codetether-agent
    cargo build --release
    # Binary at ./target/release/codetether
    ```
  </Tab>
  <Tab title="Docker">
    ```bash
    docker run -it ghcr.io/codetether/agent:latest
    ```
  </Tab>
</Tabs>

## Step 1: Configure HashiCorp Vault

CodeTether uses HashiCorp Vault for secure secrets management. No API keys are accepted via environment variables.

<AccordionGroup>
  <Accordion icon="key" title="Set up Vault connection">
    ```bash
    # Set Vault connection (these are the only env vars needed)
    export VAULT_ADDR="https://vault.example.com:8200"
    export VAULT_TOKEN="hvs.your-vault-token"
    ```
  </Accordion>
  <Accordion icon="lock" title="Store API keys in Vault">
    ```bash
    # Store provider API keys
    vault kv put secret/codetether/providers/openai api_key="sk-..."
    vault kv put secret/codetether/providers/anthropic api_key="sk-ant-..."
    vault kv put secret/codetether/providers/google api_key="AIza..."
    ```
  </Accordion>
</AccordionGroup>

<Tip>
  For development, you can run a local Vault server:
  ```bash
  vault server -dev
  export VAULT_ADDR="http://127.0.0.1:8200"
  export VAULT_TOKEN="root"
  ```
</Tip>

## Step 2: Connect to CodeTether Platform

The default mode runs as an A2A worker connecting to the CodeTether platform:

```bash
# Connect to the platform
codetether --server https://api.codetether.run

# With authentication
codetether --server https://api.codetether.run --token your-worker-token

# With a custom worker name
codetether --server https://api.codetether.run --name "my-laptop"
```

You can also use environment variables:

```bash
export CODETETHER_SERVER="https://api.codetether.run"
export CODETETHER_TOKEN="your-worker-token"
codetether
```

## Step 3: Verify It's Working

Once connected, you should see:

```
2026-02-02T12:00:00Z  INFO codetether: Connected to HashiCorp Vault
2026-02-02T12:00:00Z  INFO codetether: Connecting to A2A server at https://api.codetether.run
2026-02-02T12:00:00Z  INFO codetether: Worker registered: my-laptop
2026-02-02T12:00:00Z  INFO codetether: Waiting for tasks...
```

## Alternative: Interactive TUI

For local development, you can use the interactive terminal UI:

```bash
# Start TUI in current directory
codetether tui

# Start in a specific project
codetether tui /path/to/project
```

## Alternative: Non-Interactive Mode

Run a single command without the TUI:

```bash
# Execute a coding task
codetether run "implement a binary search function"

# Continue from last session
codetether run --continue "add unit tests"
```
