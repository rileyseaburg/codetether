## Overview

CodeTether Agent is a high-performance AI coding agent written in Rust. It runs locally on your machine and connects to the CodeTether platform as an A2A (Agent-to-Agent) worker.

## How It Works

```
┌─────────────────────────────────────────────────────────┐
│                   CodeTether Platform                   │
│                  (A2A Server)                           │
└────────────────────────┬────────────────────────────────┘
                         │ SSE/JSON-RPC
                         ▼
┌─────────────────────────────────────────────────────────┐
│                   codetether (agent)                    │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐     │
│  │ A2A     │  │ Agent   │  │ Tool    │  │ Provider│     │
│  │ Worker  │  │ System  │  │ System  │  │ Layer   │     │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘     │
│                                                         │
│  ┌──────────────────────────────────────────────────┐   │
│  │              HashiCorp Vault                     │   │
│  │           (Secrets Management)                   │   │
│  └──────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

1. **Worker Mode** (default): Connect to the CodeTether platform and process tasks
2. **TUI Mode**: Interactive terminal interface for local development
3. **Server Mode**: Serve as an A2A agent for other agents to connect to

## Agents

The agent system provides specialized modes for different types of work:

| Agent | Access Level | Use Case |
|-------|-------------|----------|
| **Build** | Full (read/write/execute) | Development work, implementation |
| **Plan** | Read-only | Analysis, exploration, planning |
| **Explore** | Read-only | Code navigation, discovery |

## Tools

Built-in tools for interacting with your codebase:

| Tool | Description |
|------|-------------|
| `read_file` | Read file contents |
| `write_file` | Create or overwrite files |
| `list_dir` | List directory contents |
| `glob` | Find files matching patterns |
| `grep` | Search with regex |
| `edit` | Apply search/replace patches |
| `bash` | Execute shell commands |

## Operating Modes

### Default: A2A Worker

```bash
codetether --server https://api.codetether.run
```

Connects to the platform and waits for tasks.

### Interactive TUI

```bash
codetether tui
```

Opens an interactive terminal interface for local development.

### Non-Interactive

```bash
codetether run "implement feature X"
```

Runs a single task and exits.

### API Server

```bash
codetether serve --port 4096
```

Starts an HTTP server for API access.

## Requirements

- **Operating System**: Linux, macOS, or Windows
- **Rust**: 1.85+ (for building from source)
- **HashiCorp Vault**: Required for secrets management
- **AI Provider**: At least one provider API key in Vault
