# Artifacts

Generated outputs and historical binary assets that are not core source code.

- `releases/` — previously generated Helm/chart packages.
- `audio/` — sample audio/PCM recordings.
- `downloads/` — downloaded third-party packages.
- `misc/` — scratch PRDs/progress files and other one-off generated files.

Prefer GitHub Releases, object storage, or Git LFS for large/generated artifacts going forward.
